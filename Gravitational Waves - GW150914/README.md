# GW150914 – First gravitational‑wave detection 🔭

This project processes open LIGO data around the GW150914 event. The pipeline includes:

- Bandpass filtering (20–300 Hz)
- Whitening (frequency‑domain normalisation)
- Q‑transform spectrogram (`gwpy`)

The final output clearly shows the characteristic **chirp** – a curved, ascending track in the time‑frequency plane – confirming the detection.
