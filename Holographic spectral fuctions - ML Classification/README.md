In the AdS/CFT correspondence, the spectral function ρ(ω) encodes the response of the dual field theory.  
- **Black hole (BH)** → continuous spectrum with a broad quasi‑normal peak (deconfined plasma).  
- **Thermal AdS (Thermal)** → discrete set of narrow peaks with a low‑frequency gap (confined phase).

This project generates synthetic ρ(ω) for both classes, extracts physically motivated features, and trains classifiers to identify the underlying geometry.
